Cookiecutter-datascience
========================

Cookiecutter template for datascience.

Usage
-----
To start a new project from this cookiecutter, install cookiecutter _(duh)_ and then do;

```bash
cookiecutter gh:eliavw/cookiecutter-datascience
```
